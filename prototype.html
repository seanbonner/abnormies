Presets

How to read this: Inputs map to on-chain readable signals. Whisper count = source Transfer events. Customizations = setTransformBitmap event count. Bleeds + Tremors = Abnormies contract event log. Everything else (position seeding, XOR cancellation, agent inversion) is render-time computation. Sealed and Fossil state freeze the canvas but don't change visual; not modeled here. Sealing target is constrained to non-self by contract rule (not visible at the single-token prototype level).
